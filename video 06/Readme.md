# 📌 SEO (Search Engine Optimization) in HTML

---

## 📖 Introduction
**SEO (Search Engine Optimization)** is the process of improving a website so that it ranks higher in search engine results like Google.  

In HTML, SEO mainly focuses on using proper tags and structure so search engines can easily understand your content.

👉 **Why SEO is important?**
- Increases website visibility 🔍  
- Brings more organic (free) traffic 🚀  
- Improves user experience 👍  

---

## ✨ Key Features / Concepts

| Concept | Description |
|--------|------------|
| **Meta Tags** | Provide information about the webpage |
| **Title Tag** | Defines page title shown in search results |
| **Heading Tags** | Structure content (H1–H6) |
| **Alt Attributes** | Describe images for search engines |
| **Semantic HTML** | Uses meaningful tags like `<header>`, `<article>` |
| **URL Structure** | Clean and readable URLs |

---

## 🧠 Syntax / Basic Structure

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <title>My SEO Friendly Page</title>
    <meta charset="UTF-8">
    <meta name="description" content="This is a SEO optimized webpage">
    <meta name="keywords" content="HTML, SEO, Web Development">
    <meta name="author" content="Your Name">
</head>
<body>
    <h1>Main Heading</h1>
    <p>Content goes here...</p>
</body>
</html>
